# How to Add the Google Dorks Blog Post

## Option 1: Using the Admin Interface (Recommended)

1. **Navigate to the Admin Panel**
   - Go to `/admin/createblog` in your browser
   - Make sure you're logged in as an admin user

2. **Fill in the Form**
   - Use the content from `google-dorks-blog-content.md`
   - Or use the "Paste from ChatGPT" feature with the formatted content below

3. **Submit the Blog**
   - Review all fields
   - Click "Create Blog"
   - The blog will be published automatically
